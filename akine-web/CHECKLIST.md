# Frontend checklist

Antes de dar una tarea por terminada verificar:
- no hay imports rotos
- no hay servicios inexistentes
- no hay dependencias faltantes
- el tipado compila
- formularios siguen válidos
- no se rompió routing
- no se cambió diseño fuera del alcance
- loading/empty/error state sigue coherente
- build/lint considerados
- riesgo manual explicitado
